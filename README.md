# Expense Tracker

シンプルな支出管理アプリです。  
日々の支出を入力して一覧で確認できます。

---

## 機能

- 支出の追加
- 支出の編集
- 支出の削除
- カテゴリでフィルター
- 合計金額の表示

---

## 使用技術

- React
- useState
- JavaScript

---

## 使い方

1. 支出名、金額、カテゴリ、日付を入力
2. 「Add Expense」をクリック
3. 一覧に表示される
4. Editで編集、Deleteで削除できる
5. カテゴリを選ぶとフィルターできる

---
